## Overview

Structured Prompt Builder is a browser-first prompt creation tool built to make prompt design easier to shape, review, and keep organized. It pairs a visual editor with live previewing and prompt organization so you can iterate quickly while keeping everything structured and easy to manage.

## FAQ

**How do I update the app?**  
Download the latest build from the project page and replace your current copy, or follow the repository workflow if you are running it from source.

Make sure your browser supports the required frontend features, then refresh the page or clear local app data if needed.

**Can I change the configuration?**  
Yes. Check the app settings or local persistence options in the browser-based interface and adjust them to fit your workflow.
